StudentApp.java

The main class responsible for:

Taking user input for the number of students.

Creating and storing student objects in an array.

Asking whether the user wants to print all students' report cards.

_______________________________________________________________________

Student.java (in the dto package)

A Data Transfer Object (DTO) class containing:

inputDetails() – Takes input for student name, roll number, and marks.

calculateTotal() – Computes total marks for the student.

calculateAvg() – Calculates the average marks.

getGrade() – Returns the grade based on the average.

printReportCard() – Displays the full report card including name, roll number, marks, total, average, and grade.
